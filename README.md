# misc-plugin

Miscellaneous skills for Claude Code - terminal UI design & copywriting.

## Installation

```bash
/plugin install misc
```

Or add the marketplace first:
```bash
/plugins marketplace add iamladi/cautious-computing-machine
/plugin install misc
```

## Skills

### terminal-ui-designer

Create distinctive, production-grade terminal user interfaces. Use when building CLI tools, TUI applications, or terminal-based interfaces.

Covers:
- Box drawing & borders (single, double, rounded, heavy, ASCII, block)
- Color & theme strategies (ANSI 16, 256, true color, monochrome)
- Typography & text styling (ASCII art, text weight, Unicode symbols)
- Layout & spatial composition (panels, columns, tables, whitespace)
- Motion & animation (spinners, progress bars, transitions)
- Data display (sparklines, bar charts, trees, status indicators)
- Anti-patterns to avoid

### copywriting

Direct response copywriting framework for landing pages, marketing copy, and product descriptions.

**THE SLIDE** (7-step formula):
1. Headline (80% of work)
2. Problem (quantify it)
3. Agitate (make vivid)
4. Credibility
5. Solution (transform)
6. Proof (specific)
7. CTA (benefit)

**SO WHAT? CHAIN**: Feature → Functional → Financial → Emotional (write from emotional end)

**RHYTHM**: Short. Then breathe. Land it.

**SPECIFICITY**: Replace vague with concrete ("lots of money" → "$47,329")
